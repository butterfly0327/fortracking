# Raspberry Pi Fan (Servo) Control Spec v1.9 Implementation

## 구현 성공 항목
- 10Hz JSON 수신에서 `ex/sx/ev/sv`를 최신 슬롯으로 저장하고, `none` 및 범위(0~1000) 검증을 적용했습니다.
- 대표 좌표/속도 계산 규칙과 정규화(0~1) 규칙을 구현했습니다.
- 50Hz 제어 루프에서 워치독(RX_TIMEOUT), 좌표 EMA, 데드밴드, 방향 반전 게이트/브레이크 로직을 적용했습니다.
- 속도 EMA 기반 3단 기어(LOW/MID/HIGH)와 안정화 카운트, 램프(가감속) 로직을 구현했습니다.
- 서보 각도 clamp(10°~170°) 및 끝각 보호를 구현했습니다.

## 실패/미구현 항목
